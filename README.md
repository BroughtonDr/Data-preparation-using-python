## 📋 Overview

This notebook walks through a complete data preparation pipeline for a hypertension dataset of **52,350 Indonesian patients**. The goal is to produce a clean, analysis-ready dataset for predicting changes in systolic and diastolic blood pressure.

## 🚶 Pipeline Steps

### Step 1 — Import Libraries
Loads all required Python libraries.

### Step 2 — Open the Dataset
Reads `hypertension_52350.csv` into a pandas DataFrame.

### Step 3 — Know the Data
Inspects shape, data types, and column names to understand the structure before any cleaning.

### Step 4 — Visualise Missing Values
Uses `missingno` matrix and heatmap to identify patterns and correlations in missing data across columns.

### Step 5 — Missing Value Percentage
Calculates the percentage of missing values per column, sorted in descending order, to prioritise which columns need attention.

### Step 6 — Drop Rows with Missing Values
Applies `dropna()` as a first-pass filter to remove records with critical missing fields.

### Step 7 — Select Variables
Narrows the dataset to 22 clinically and analytically relevant columns:

> `Diagnosis`, `Age`, `AGE of HTN`, `Sex`, `BMI`, `SBP`, `DBP`, `Smoking/Naswar`, `Physical Activity`, `Treatment`, `Family HTN`, `Family CVD`, `Diet plan`, `meals/day`, `Income`, `Job`, `Headache`, `Dizziness`, `Chest Pain`, `Palpitations`, `Province`, `Date of Visit`

### Step 8 — Convert to Numeric
Coerces `SBP`, `DBP`, `BMI`, `Age`, and `AGE of HTN` to numeric types, handling any non-numeric values with `errors='coerce'`.

### Step 9 — Make Categorical Variables
Converts 13 nominal/ordinal columns (e.g. `Sex`, `Diagnosis`, `Treatment`) to pandas `category` dtype for memory efficiency and correct downstream handling.

### Step 10 — Convert Date Column
Parses `Date of Visit` into `datetime64` format.

### Step 11 — Fill Missing Values
Imputes remaining missing values in `SBP`, `DBP`, and `BMI` with their respective medians (robust to outliers).

### Step 12 — Descriptive Statistics
Generates summary statistics for both numeric (`.describe()`) and categorical (`.describe(include='category')`) variables.

### Step 13 — GroupBy Analysis
Computes mean `SBP`, `DBP`, and `BMI` by **Sex**, and mean `SBP`, `DBP`, and `Age` by **Diagnosis** category.

### Step 14 — Cross Tabulation
Produces frequency tables for:
- Sex × Diagnosis (counts)
- Smoking Status × Diagnosis (row-normalised proportions)

### Step 15 — Duplicate Detection & Removal
Identifies and removes exact duplicate rows using `drop_duplicates()`.

### Step 16 — Outlier Detection
Generates side-by-side boxplots for `SBP`, `DBP`, and `BMI` to visually flag extreme values.

### Step 17 — Merge with Province Mapping
Left-joins the dataset with `province_mapping.csv` on `Province` to enrich each record with `Region`, `Island`, `Population_million`, and `Health_Facility_per_100k`.

### Step 18 — Descriptive Graphs
Produces four visualisations:
- Distribution of SBP and DBP (histograms)
- Hypertension diagnosis count by Sex (bar chart)
- Smoking status distribution (bar chart)
- Physical activity level breakdown (pie chart)
- Mean SBP by Province (horizontal bar chart)

### Step 19 — Save Clean Dataset
Exports the final cleaned DataFrame to `hypertension_clean.parquet` for efficient storage and fast loading in downstream modelling steps.
