str — A string is text, like a word or sentence, written inside quotes.
int — An integer is a whole number with no decimal point, like 5 or -100.
float — A float is a number with a decimal point, like 3.14 or 9.99.
bool — A boolean is simply a yes or no value — either True or False.
list — A list is an ordered collection of items that you can change, like a shopping list.
tuple — A tuple is like a list, but once you create it, you cannot change it.
dict — A dictionary stores information as pairs of labels and values, like a contact card with a name and phone number.
set — A set is a collection that automatically removes any duplicate values, keeping only unique items.
